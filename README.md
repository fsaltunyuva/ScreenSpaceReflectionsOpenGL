# ScreenSpaceReflectionsOpenGL

Screen Space Reflection (SSR) implementation in OpenGL using the Sibenik Cathedral scene created by Marko Dabrovic.

Sibenik Cathedral is a popular 3D model used for testing and demonstrating various rendering techniques, including reflections. It includes 75284 triangles and 83490 vertices. As stated in [this copyright notice](https://github.com/fsaltunyuva/ScreenSpaceReflectionsOpenGL/blob/main/sibenik/copyright.txt), model is created by Marko Dabrovic, with holes corrected by Kenzie Lamar at Vicarious Visions and high-resolution texture and bump maps painted by Morgan McGuire.
